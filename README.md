# Website for Article Registration v1.1

Основной проект находится в папке `site-version-1.0`.

## Запуск

Открой терминал в папке проекта:

```powershell
cd .\site-version-1.0
```

Установи зависимости:

```powershell
npm.cmd install
```

Запусти сайт:

```powershell
npm.cmd run dev
```

После запуска открой:

```text
http://localhost:3000
```

Админка доступна по адресу:

```text
http://localhost:3000/login
```

Стандартные данные для входа:

- логин: `admin`
- пароль: `ChangeMeAdmin123!`
