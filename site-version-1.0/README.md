# Site Version 1.0

Основная рабочая локальная версия сайта.

## Запуск

Открой терминал в этой папке:

```powershell
cd "C:\Users\Григорий\OneDrive\Dokumente\Website-for-article-registration\site-version-1.0"
```

Запусти сайт:

```powershell
npm.cmd run dev
```

После запуска открой:

```text
http://localhost:3000
```

## Важно

- Не открывай `html`-файлы напрямую, сайт запускается через Next.js.
- Если в PowerShell команда `npm` ругается на `npm.ps1`, используй `npm.cmd`.
- Для стабильного локального запуска в этой версии используется `webpack`.
